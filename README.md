# @heusalagroup/fi.hg.create

Sub module for [project templates](https://hg.fi/#project-templates).

### Installation

Using [`hgm`](https://github.com/heusalagroup/hgm):

```shell
hgm i fi.hg.core
hgm i fi.hg.create
```
